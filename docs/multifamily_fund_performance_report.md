## Fund Performance Metrics Definitions and Benchmarking

### Key Metrics Explanation:
- **IRR (Internal Rate of Return)**: Annualized return accounting for timing of cash flows
- **TVPI (Total Value to Paid-In)**: (Distributions + Residual Value) ÷ Paid-In Capital
- **DPI (Distributions to Paid-In)**: Cumulative distributions ÷ Paid-In Capital  
- **RVPI (Residual Value to Paid-In)**: Current portfolio value ÷ Paid-In Capital
- **NAV**: Net Asset Value of remaining investments

### Performance Quartile Breakdowns:
- **First Quartile**: Top 25% performers (≥75th percentile)
- **Second Quartile**: 50th-75th percentile 
- **Third Quartile**: 25th-50th percentile
- **Fourth Quartile**: Bottom 25% performers (≤25th percentile)

## Benchmarking Framework for Fund Managers

### Performance Evaluation Methodology:
1. **Compare to appropriate vintage year peer group**
2. **Adjust for strategy (core, value-add, opportunistic)**  
3. **Consider geographic focus and market timing**
4. **Evaluate against both private and public market alternatives**
5. **Assess risk-adjusted performance using multiple metrics**

### Quartile Performance Identification:
**How to Use the Tables**:
- **Top Quartile**: Performance at or above "Upper Quartile" metrics
- **Second Quartile**: Between Median and Upper Quartile performance
- **Third Quartile**: Between Lower Quartile and Median performance  
- **Bottom Quartile**: At or below "Lower Quartile" metrics

### Fee Impact Analysis:
**Typical Fee Structures**:
- **Management fees**: 1.5-2.5% of committed capital
- **Carried interest**: 15-25% of profits above preferred return
- **Total cost impact**: 200-400 basis points drag on returns
